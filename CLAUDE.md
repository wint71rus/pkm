## Про пользователя
Александр Манипов, системный администратор. Windows основной ПК (PowerShell), сервера Linux. Уровень Linux начальный, активно развивается. Язык общения — **русский**.

Полный профиль и стек: [memory/context/facts.md](memory/context/facts.md).

## Как отвечать
- На русском, кратко и по делу
- Код давать сразу рабочий, без заглушек
- Объяснять, что делает код (пользователь учится попутно)
- Если несколько способов — сначала самый простой
- Напоминать про типичные ошибки новичков в Linux

## Shell / окружение
- Основной shell: **PowerShell** (Windows 11) — используй команды PowerShell, а не bash, если явно не указано иное
- На серверах: bash
- Git remote: `git@github.com:wint71rus/pkm.git`

## Автосинхронизация — ВАЖНО
В корне лежит [sync.ps1](sync.ps1). Scheduled Task **«PKM-AutoSync»** автоматически делает **локальный коммит** каждые 30 мин. Пуш на remote — **только вручную** (`.\sync.ps1 -Push`).

**Для AI-агента**: не создавай коммиты сам, если только пользователь явно не попросил. Локальный автокоммит справится. Не делай `git push` без явного запроса — это видимое остальным действие.
