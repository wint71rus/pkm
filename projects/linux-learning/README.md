# Linux Learning

> Создано: 2026-04-24
> Статус: активен

## Цель
Выйти на уверенный уровень администрирования Linux-серверов: базовые команды, systemd, пакетные менеджеры, сеть, логи, bash-скрипты.

## Темы к освоению
- [ ] Навигация и файловая система (ls, cd, find, tree, permissions)
- [ ] Пакетные менеджеры (apt, dnf/yum)
- [ ] systemd (systemctl, journalctl, unit-файлы)
- [ ] Пользователи и права (useradd, sudo, chmod, chown)
- [ ] Сеть (ip, ss, firewall — ufw/firewalld/iptables)
- [ ] Процессы и логи (ps, top/htop, tail, journalctl)
- [ ] Cron / systemd timers
- [ ] SSH (ключи, конфиг, пробросы портов, ssh-agent)
- [ ] Bash-скрипты (переменные, условия, циклы, функции)
